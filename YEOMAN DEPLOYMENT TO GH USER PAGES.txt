[INIT]

new local folder
yeoman init
grunt build
gitignore => rm dist
git init
git remote add origin [repo]
git add dist
git commit -m "init dist"
git subtree push --prefix dist origin master
git checkout -b src
git add .
git commit -m "init src"
git push origin src

You might get an error like this Updates were rejected because the tip of your current branch is behind. You can solve this by force pushing to the remote (be careful though, it will destroy whatever is already there).
=>
git push origin `git subtree split --prefix dist master`:master --force
